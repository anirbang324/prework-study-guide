Prework Study Guide

HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It structures the content on the web and consists of elements represented by tags.

CSS

CSS (Cascading Style Sheets) is used for styling web pages. It defines how HTML elements should be displayed, including layout, colors, and fonts.

Git

Git is a distributed version control system that tracks changes in source code during software development. It allows collaboration among developers and helps in managing project history.

JavaScript

JavaScript is a programming language that enables interactive web pages. It can manipulate the Document Object Model (DOM) and add dynamic behavior to websites.
